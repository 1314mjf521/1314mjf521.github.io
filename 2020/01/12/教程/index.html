对于纯小白(windows也不会装,非电脑行业)重要提示：
不知道EFI是什么，PE是什么，BIOS是什么，不愿意去了解，不懂，怕麻烦，嫌烦，就不要浪费精力了，不像装windows那么容易，需要花费大量时间和耐心，不排除运气好的一装就可以装上，但是之后的驱动也是要自己多学习，多试验的。不是嘲讽，而是爱莫能助，不要浪费了时间得不偿失了。

安装前的问题：

  1. 没有耐心，不爱学习钻研的解决办法：
    不安装，因为没有人有精力一步一步指导你安装，安装后也需要根据自己的机型配置自己调试，大家都要工作上班和休息，因为没有基础问的一些问题请先了解，让了解不去了解硬着头皮装的，就装吧。

  2. 我这配置能装吗？：
    只要不是太老的配置基本都还行，太老的设备问题较多没有耐心就不要安装了

  3. 确认下自己的BIOS支持UEFI吗？
    不知道什么是UEFI可以百度查一下

  4. 我这个显卡免驱吗？能装吗？
    黑果小兵的显卡兼容列表：
    https://blog.daliansky.net/Mojave-Hardware-Support-List.html
    硬件兼容列表：
    http://uee.me/cQVYK

  5. 有这个机子的EFI吗？
    远景：http://bbs.pcbeta.com
    黑苹果各机型EFI整理清单：http://uee.me/cQWax

    tonymacx86: www.tonymacx86.com

开始准备安装

  1. 黑果小兵下载镜像 https://blog.daliansky.net/
    点击 分类 -> 镜像
    台式机：
    Nvdia显卡选10.13.6(17G65)正式版(别选特别版，个人经验)
    A卡看上面兼容列表，不免驱的提前搜下如何驱动

    笔记本(大部分笔记本无法使用独显)：
    核显按说都支持，部分型号可能需要下载对应驱动
    较老的型号如hd3000安装15系统需要拿出低版本系统自带的hd3000驱动文件，参考：
    http://heipingguowu.top/2019/11/02/11074/
    或黑果小兵办法：

    解决显卡存在的一些问题如黑屏卡顿，驱动下载(来自黑苹果屋)：
    http://heipingguowu.top/category/hpgw/bbqd/xkqd/
    安装视频（没有基础的请别快进，不懂的就去查，视频中讲到的不重复回答）：
    https://www.bilibili.com/video/av58097219

  2. 分区时DiskGenius提示下图：

    这个说明这个磁盘空间已经全部被分区占满了，得先缩小一个分区，腾出来一点空间。
    随便找一个分区->右键->调整分区大小->把调整后的分区 填的小一点 ->确定
    然后在这个分区->右键->新建ESP分区->确定

安装中问题：

  1. 安装不上，可以整体更换clover文件尝试或者更换小兵镜像自带的config文件（在clover/config-other）(不会更换clover看下面视频链接)，或者搜索报错英文是什么意思。我也没啥好办法小白一个能力有限，我的机器比较好装，没有实际解决过一些问题。

  2. 我有空的话可以提供你的笔记本型号帮忙查找EFI。

  3. 开机报错 This version of Mac OS X is not supported on this platform!
    添加clover的启动参数 ： -no_compat_check
    或者在clover界面按O，找到smbios，第一行敲回车修改机型 MacBookPro14,3或者更新的机型，
    不会操作的看clover是什么

  4. 卡在Couldn‘t allocate runtime area：
    更换USB2.0口尝试 http://uee.me/cQWcR

  5. 卡在Please go to https://panic.apple.com to report this panic：
    尝试更换Lilu的版本 http://uee.me/cQWcV

  6. 更换clover时用DiskGenius删除自己EFI文件夹里的clover，再扔进去其他的：
    https://www.bilibili.com/video/av61869615?from=search&seid=16610186575330581387

  8. 个人群内总结：
    1) 安装中抹掉磁盘失败的话，可以看下那块磁盘上有没有esp分区，没有的话象征性的创建一下（因为有时候其他盘已经有可用的esp分区了），或者是esp分区不够大
    2)卡加号、卡++：直接狂敲回车试试，或者将drivers64UEFI下的OsxAptioFixDrv-64.efi删除，再将OsxAptioFix2Drv-64.efi放入drivers64UEFI尝试一下，不行百度搜卡++<
    3) 禁行标志（圆圈斜杠）：删除CLOVER/kexts/Other里面的USBInjectAll.kext的驱动尝试
    4) 卡在下图EC

    下载文件如下图

    放在EFI/CLOVER/ACPI/Patched下面试试
    5) macOS未能安装在您的电脑上_安装资源已过期。退出安装器以重新启动电脑，然后再次尝试。

    6) 抹掉磁盘提示容量不如是esp分区不够大，大于300兆即可。扩容请看b站视频
    https://www.bilibili.com/video/av63894908?from=search&seid=16610186575330581387
    7) 黑果小兵的镜像自带的clover有很多config.plist文件，在clover引导界面按o可以切换，多试试，可以选择自己的核显试试。
    8) 卡在 IOConsoleusers:gIOScreenLockState3,hs0,bs0,now0,sm0x0 ，笔记本可以下载文件
    放到
    台式机在clover安装项按空格勾选 disable_nv=1 试下，或者bios禁用下核显试试
    ，或者在bios中禁用独显
    以下来自黑果小兵：https://blog.daliansky.net/Common-problems-and-solutions-in-macOS-Mojave-10.14-installation.html
    原因是系统无法识别出你的显卡驱动，临时的解决方法是：
    取消勾选Inject Intel
    或者将platform-id修改为0x12345678
    两种方法二选一，不知道如何操作的请移步Clover使用教程

    安装完成后再通过Clover Configurator设置正确的FakeID和ig-plaform-id驱动你的显卡。
    9) 应用副本已损坏，不能用来安装：点击终端改时间，看下图

    图中只是示意，改成 date 093018292019.20 或者 date 032208102015.20
    执行就可以，不可以的话，时分改成当下时间
    记得断网，并且改完点击左上角菜单里的退出终端，要不安装页面可能弹不出来
    10) Windows进不去了，看b站视频
    https://www.bilibili.com/video/av62847371
    11) pm981 别瞎凑热闹。三星PM981(a)的老铁们请绕道食用。
    12) 部分黑屏、卡进度条、跑代码后重启的问题
    黑苹果屋教程：http://heipingguowu.top/2019/10/21/2009/

安装后的问题：

安装完先看视频https://www.bilibili.com/video/av58597462

  1. EasyUEFI添加clover引导后，开机还是直接进win，clover引导项变第二或消失：
    EasyUEFI最好使用企业版，找企业破解版的
    可以在EasyUEFI里试试把windows的引导删除，再重新创建，然后把clover放到第一个位置
    如果不行的话，有些机型需要去bios里添加引导项（群友的办法）
